h1. 关于

  Rubyist是一个多人协作的群体Blog，旨在为国内开发者提供新鲜和高质量的Ruby文章

h1. 特性

* 支持BBcode
* 支持Textitle
* 代码高亮
* 支持Gist
* 支持Avatar头像
* 可以方便部署到Heroku
* Github登录
* 嵌套回复

h1. Heroku配置

heroku config:add RUBYIST_EMAIL=email RUBYIST_PASSWORD=password RUBYIST_HOST=host
heroku config:add RUBYIST_GITHUB_KEY=key RUBYIST_GITHUB_SECRET=secret
